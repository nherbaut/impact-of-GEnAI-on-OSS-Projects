  title: An IoT-Enabled Framework with Real-Time Logic Gate Validation and Equipped Large Language Model
  authors: SIMAR SINGH RAYAT, Sujal Thapa, Chandrdaeep Bhatt, Vikrant Sharma, Susheela Dahiya, Rohan Verma
  abstract: Traditional logic gate simulators mainly offer a graphical design environment while they usually do not offer
    real time interfacing to hardware devices and are still restricted for their use in educational applications. To solve
    all these limitations, we introduce PLUX, a single unified framework of automatic logic gate simulation that combines
    on-line Internet of Things (IoT) hardware communication, Artificial Intelligence (AI) based on circuit optimization, and
    code generation to digital designs. PLUX is a class of hardware-in-the-loop debugging system that permits simulation engineers
    to easily integrate hardware with software, students/researchers to simulate advanced circuits, hardware grippers/developers
    to validate circuits in context and verify performance in the real world. The framework operates with microcontroller
    platforms such as Arduino, Raspberry Pi, ESP32, and Micro Python, and can provide live feedback, fault detection, adaptive
    optimization, and is able to string of generic and self-written certain program branches. Bekritz's flexible and scalable
    architecture allows for long-term adaptability to emerging technologies, while AI algorithms embedded in the design minimize
    design inefficiencies as well as debugging time. The framework was evaluated on publicly available benchmark datasets
    as ISCAS-85 and ISCAS-89 benchmarking circuits, MCNC Benchmark suite, EPFL combinational benchmark suite and open source
    IoT traffic datasets. Experimental results show the framework achieves up to 35% faster simulation speed and 20% less
    CPU utilization compared to the normal simulators with a Transmission Reliability Index (TRI) of 0.96, for USB communication
    and other communication means like Bluetooth and Wi-Fi communication.
