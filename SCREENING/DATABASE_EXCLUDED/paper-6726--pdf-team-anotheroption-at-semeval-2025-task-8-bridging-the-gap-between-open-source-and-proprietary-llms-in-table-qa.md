  abstract: 'This paper presents a system developed for SemEval 2025 Task 8: Question Answering (QA) over tabular data. Our
    approach integrates several key components: text-to-SQL and text-to-code generation modules, a self-correction mechanism,

    achieved an accuracy of 80%, resulting in a top-13 ranking among the 38 participating teams. Our pipeline demonstrates
    a significant improvement in accuracy for open-source models and achieves a performance comparable to proprietary LLMs
    in QA tasks over tables. The code is available at GitHub repository.'
