  title: 'ServiceCoder: A Service Computing Code Generation Model'
  authors: Yue Chen, Lei Yu, Xiang Zhou, Haiquan Su, Junhua Liu, Yuhang Wang
  abstract: This paper reports on work in progress on the application of a Parameter-Efficient Fine-Tuning (PEFT[1]) that
    combines a baseline code generation model (CGM) with Quantization and Low-rank Adapter (QLoRA[2]). Thereby improving the
    progress of work on baseline models in dealing with the quality of service code. Current research has focused on applying
    CGM to code generation tasks for common programming projects. At the same time, fewer attempts have been made to fine-tune
    the models for service computing, resulting in poor performance of open-source CGMs for code generation tasks in this
    domain. By acquiring a dataset of code snippets related to service computing tasks, we develop a CGM, ServiceCoder. Experimental
    results show that ServiceCoder is better than pre-trained CGMs in programming in the service computing domain. Finally,
    the RESTful[3] API integration ensures the security and convenience of the model in real applications, offering a novel
    approach to investigating the intersection of code generation and Service Computing; thus, this work makes an exploratory
    contribution.
