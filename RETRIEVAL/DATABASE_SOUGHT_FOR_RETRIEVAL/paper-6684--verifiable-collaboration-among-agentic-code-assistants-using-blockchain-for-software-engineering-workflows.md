  title: Verifiable Collaboration among Agentic Code Assistants using Blockchain for Software Engineering Workflows
  authors: Preet Saumil Merchant, S.P. Gaba
  abstract: As generative AI becomes increasingly integrated into software engineering, the emergence of autonomous coding
    agents introduces critical challenges around accountability, trust, and collaborative transparency. In this work, we propose
    a novel framework that combines agentic AI with blockchain technology to enable verifiable, auditable, and decentralized
    collaboration among autonomous code assistants. Our system models software development tasks, such as code generation,
    review, testing, and documentation, as multi-agent workflows, where each agent operates with autonomy while maintaining
    a shared project context. Every action performed by an agent, whether it’s submitting code, flagging an issue, or approving
    a merge, is immutably recorded on a blockchain ledger, allowing teams and stakeholders to trace the provenance of software
    artifacts and decisions. This cryptographically verifiable audit trail fosters trust in agent contributions, enhances
    collaborative governance, and ensures reproducibility in AI-assisted software engineering pipelines. We present a prototype
    implementation using open-source agent orchestration tools and a lightweight blockchain backend to simulate common developer
    scenarios. We also discuss the implications of blockchain-backed agent collaboration for regulatory compliance, secure
    software supply chains, and the future of decentralized software engineering practices.
