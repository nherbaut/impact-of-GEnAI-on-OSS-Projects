  title: 'Archelon AI: Programming Assistant for Legacy Codebases'
  authors: T. T. L. Denuri, S. Ragukanthan, W. P. Anjalee Shalika Wijesekara, M. D. Prageeth, A.T. P. Silva, L. Attanayaka
  abstract: 'Maintaining and upgrading legacy software systems poses a growing challenge due to poor documentation, increased
    system complexity, and a shortage of developers skilled in older languages like Perl. This paper presents Archelon AI,
    a domain specific programming assistant designed to improve developer productivity and system maintainability. The system
    comprises of two main components: a chat assistant and an IDE integration plugin for Visual Studio Code. Archelon AI is
    powered by a fine-tuned Qwen2.5 Coder 3B instruct model on the Perl Programming Language. It supports natural language
    prompts for code generation, explanation, and error debugging. The system enhances response and accuracy by incorporating
    relevant context from user-provided PDFs, web links, and GitHub repositories using Retrieval Augmented Generation (RAG)
    with vector databases. Along with context-awareness, the chat assistant supports session-based interactions which enhances
    the conversation organization and continuity. The IDE plugin supports context-aware code suggestions, ghost text generation,
    alternative implementations, a real-time error detection. Together these features deliver accurate, context-sensitive
    assistance that streamlines development workflows in outdates codebases. Archelon AI demonstrates the potential of integrating
    large language Models (LLMs) and modern AI infrastructure to support legacy programming environments effectively.'
