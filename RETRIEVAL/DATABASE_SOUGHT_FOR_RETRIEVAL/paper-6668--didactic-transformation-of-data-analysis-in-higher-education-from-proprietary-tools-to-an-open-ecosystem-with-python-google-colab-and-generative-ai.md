  title: 'Didactic Transformation of Data Analysis in Higher Education: From Proprietary Tools to an Open Ecosystem with Python,
    Google Colab and Generative AI'
  authors: Milin Erica, Quiroga Silvia, Flecha Rubén
  abstract: This work-in-progress reports a didactic transition from proprietary software (Microsoft Excel and EasyFit) to
    an open, cloud-based ecosystem built on Google Colab and Python for teaching end-to-end data work in an engineering “Simulation”
    course-covering data ingestion and processing, fitting probability density functions to empirical data, and simulating
    new datasets from the selected distributions. The objective is to foster technical skills aligned with current market
    demands, enhance process reproducibility, facilitate collaborative work, and leverage AI-assisted learning for improved
    efficiency. The paper describes the application context, the structure of the implemented workshop-including AI features
    like code generation and debugging-and outlines an evaluation methodology to assess its impact on learning. This transition,
    in addition to being technologically relevant, aligns with principles of openness, equity, and pedagogical renewal in
    data science education, promoting accessibility in diverse educational contexts.
