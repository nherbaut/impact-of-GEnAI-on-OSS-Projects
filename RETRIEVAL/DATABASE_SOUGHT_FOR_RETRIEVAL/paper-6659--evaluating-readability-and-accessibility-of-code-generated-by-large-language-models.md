  title: Evaluating Readability and Accessibility of Code Generated by Large Language Models
  authors: Y.P. Zakharov, Maxim Bakaev
  abstract: 'This paper investigates the readability and accessibility of Python code automatically generated by large language
    models. We evaluate two open-source instruction-tuned models, DeepSeek-R1-0528 and gpt-oss-20b-Q2, on a controlled subset
    of 15 tasks from the HumanEval+ benchmark. For each task we compare two prompting strategies: a baseline instruction and
    an enhanced prompt that enforces stylistic constraints (consistent naming, docstrings, limited nesting, and PEP8 conformance).
    Generated solutions are assessed for functional correctness with pytest and for style and maintainability using Pylint.
    Our results show that enhanced prompting substantially improves static style scores: average Pylint increased from 4.09
    to 6.41 for DeepSeek-R1-0528 and from 6.14 to 8.13 for gpt-oss-20b-Q2. We analyse common warning categories, report cyclomatic
    complexity trends, and provide qualitative examples illustrating how prompt design affects naming, documentation, and
    nesting. The study demonstrates that lightweight prompt-engineering is an effective and low-cost method to improve the
    accessibility of model-generated code, and outlines practical steps for integrating such techniques into automated code-generation
    workflows.'
